# NH Proteção Veicular

Site institucional desenvolvido para a **Novo Horizonte Proteção Veicular**, com foco em apresentar os serviços da empresa, facilitar solicitações de cotação e oferecer acesso rápido aos principais canais de atendimento.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Font Awesome
* FormSubmit

## Funcionalidades

* Layout responsivo para desktop e celular
* Página inicial institucional
* Popup de cotação com formulário
* Página de 2ª via de boleto
* Página de escritórios
* Botão flutuante de WhatsApp
* Barra flutuante de redes sociais
* Botões de ligação direta e contato

## Estrutura

```txt
├── index.html
├── boleto.html
├── escritorios.html
├── style.css
├── script.js
└── assets/
```

## Como executar

Abra o arquivo `index.html` diretamente no navegador ou utilize a extensão **Live Server** no VS Code.

## Deploy

O projeto pode ser publicado em qualquer hospedagem estática, como:

* Vercel
* Netlify
* GitHub Pages
* Hostinger
